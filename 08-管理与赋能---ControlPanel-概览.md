# Control Panel 概览

> 更新时间：2025-12-28
> 文档来源：https://palantir.com/docs/foundry/administration/control-panel/

## 概述

**Control Panel** 是 Foundry 的**集中式管理界面**，用于执行所有管理工作流。您可以从 Applications Portal 访问 Control Panel。

Control Panel 主页提供您的注册和组织概述以及执行常见管理工作流的链接。

## 核心功能

### 1. 集中式管理

- **单一界面**：所有管理工作流的集中式接口
- **组织和注册管理**：管理多个组织和注册
- **搜索功能**：快速查找特定设置
  - 使用侧面板中的 "Search" 按钮
  - 快捷键：`Cmd+J` (macOS) 或 `Ctrl+J` (Windows)

### 2. 分层设置

设置按注册/组织级别分组在侧面板中显示为选项卡：

```
┌─────────────────────────────────────────────────────────────┐
│              Control Panel 设置层级                          │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  Enrollment Level（注册级别）                                │
│  ├── Authentication（身份验证）                            │
│  ├── Retention（保留策略）                                  │
│  ├── Email administration（邮件管理）                      │
│  └── Enrollment settings（注册设置）                        │
│                                                             │
│  Organization Level（组织级别）                              │
│  ├── Resource Management（资源管理）                        │
│  ├── Organization settings（组织设置）                      │
│  └── Applications（应用程序）                               │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

## 界面导航

### 主页

Control Panel 主页显示：
- **注册概览**：您的注册概述
- **组织概览**：组织列表和状态
- **快速链接**：常见管理任务的快速链接
- **通知**：重要通知和警报

### 侧面板

侧面板按类别组织设置：
- **Enrollment 级别设置**：影响整个注册的设置
- **Organization 级别设置**：特定于组织的设置
- **快速搜索**：快速查找特定设置

## 常见任务

### 1. 用户管理

- **添加用户**：通过自助用户目录添加用户

- **多因素认证**：启用 MFA

- **异常检测**：配置异常检测

### 3. 安全配置

- **网络配置**：配置入口和出口
- **容器治理**：配置容器限制
