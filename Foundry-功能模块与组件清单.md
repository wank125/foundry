## 四、组件功能详解

### 4.1 数据连接与集成组件

#### Data Connection（数据连接）
**组件功能**：
- 支持 200+ 数据源连接器
- 提供直接连接、Agent 代理、Webhook 三种连接方式
- 支持数据库、SaaS 应用、文件系统、流数据等数据类型
- 提供增量同步和实时流处理能力

#### Pipeline Builder（管道构建器）
**组件功能**：
- 提供点击式和代码式双模式数据转换
- 自动构建路径修剪和优化
- 支持批处理和流式管道
- 提供类型安全函数和严格输出检查
- 集成 LLM 数据转换节点

#### Code Repositories（代码仓库）
**组件功能**：
- 支持 Python、Java、SQL、R 语言
- 提供 Git 工作流和分支管理
- 内置单元测试和 CI/CD 集成
- 支持项目引用和依赖管理

#### HyperAuto（超级自动化）
**组件功能**：
- SAP 自动化数据集成
- 自动生成数据管道
- 快速发现和集成 ERP/CRM 数据
- 支持实时流处理

### 4.2 模型连接与开发组件

#### AIP Logic（AIP 逻辑）
**组件功能**：
- 提供 LLM 驱动的逻辑函数
- 支持无代码/低代码 AI 逻辑构建
- 集成 Foundry Functions 和 Actions
- 提供 AI 逻辑评估和优化

#### Model Catalog（模型目录）
**组件功能**：
- 发现和管理 Palantir 提供的模型
- 支持大语言模型 (LLMs)
- 提供模型游乐场测试
- 管理模型版本和生命周期

#### Modeling Objectives（建模目标）
**组件功能**：
- 管理 ML 模型全生命周期
- 提交、发布、部署模型
- 监控模型性能指标
- 集成 CI/CD 流程

#### Code Workspaces（代码工作区）
**组件功能**：
- 集成 JupyterLab、RStudio Workbench、VS Code
- 提供安全隔离的开发环境
- 支持本体集成和模型训练
- 提供交互式数据访问

#### AIP Agent Studio（AIP 代理工作室）
**组件功能**：
- 构建交互式 AI 助手
- 配置企业特定信息和工具
- 在平台内外部署 AI 代理
- 支持 4 层代理框架

### 4.3 本体构建组件

#### Ontology Manager（本体管理器）
**组件功能**：
- 定义和管理对象类型
- 创建和维护链接类型
- 配置对象属性和值类型
- 管理结构类型和共享属性

#### Functions（函数）
**组件功能**：
- 支持 TypeScript、Python、Java 函数
- 提供对象查询和聚合
- 支持本体编辑和外部 API 调用
- 集成语义搜索和模型函数

#### Action Types（操作类型）
**组件功能**：
- 定义参数化操作
- 配置提交标准验证规则
- 支持函数驱动的复杂业务逻辑
- 提供副作用（通知、Webhook）

#### Object Explorer（对象浏览器）
**组件功能**：
- 搜索和探索对象
- SQL 分析支持
- 过滤和可视化对象
- 应用 Actions 和导出结果

#### Object Views（对象视图）
**组件功能**：
- 作为对象详情的中心枢纽
- 支持完整视图和面板视图
- 集成多种应用视图
- 提供标签页和侧边栏配置

#### Vertex（顶点）
**组件功能**：
- 图形关系分析和可视化
- 探索对象关系
- 事件和时间序列分析
- 支持场景配置

#### Map（地图）
**组件功能**：
- 地理空间数据可视化
- 点、线、面数据渲染
- 时间序列轨迹分析
- 事件时间轴显示

#### Machinery（流程挖掘）
**组件功能**：
- 流程挖掘和分析
- 从事件日志发现流程
- 定义和监控性能指标
- AIP 编排支持

#### Dynamic Scheduling（动态调度）
**组件功能**：
- 调度和资源分配工作流
- 甘特图和日历小部件
- 约束验证和推荐引擎
- 实时交互编辑

#### Foundry Rules（Foundry 规则）
**组件功能**：
- 业务规则引擎
- 基于本体的规则配置
- 调度工作流支持
- 条件逻辑和验证

### 4.4 用例开发组件

#### Workshop
**组件功能**：
- 灵活的面向对象应用构建
- 支持无代码/低代码/代码小部件
- 对象和链接驱动
