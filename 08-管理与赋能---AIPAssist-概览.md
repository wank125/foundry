# AIP Assist 概览

> 更新时间：2025-12-28
> 文档来源：https://palantir.com/docs/foundry/assist/overview/

## 概述

**AIP Assist** 是一款由 **LLM 驱动的支持工具**，旨在帮助用户导航、理解并利用 Palantir 平台生成价值。用户可以用自然语言向 AIP Assist 提问，并实时获得帮助。

## 核心价值

### 主要好处

| 好处 | 说明 |
|------|------|
| **用户友好的界面** | 由 LLM 驱动，具有直观的界面，使用户能够轻松提问并获得相关、自然且易于理解的回答 |
| **实时帮助** | 提供实时帮助，快速解决问题和疑问，提高用户生产力，减少对支持团队的依赖 |
| **多语言支持** | 可以用所有常见语言回答查询 |
| **上下文感知** | 知道您所在的 Foundry 应用程序，保持对话的上下文 |
| **Foundry 级别的安全性** | 完全尊重 Palantir 的 AI 伦理原则，不会访问您的数据 |
| **迭代改进** | 用户可以对回答质量提供反馈，帮助工具在开发过程中不断改进 |

## 访问方式

### 打开 AIP Assist

有两种方式打开 AIP Assist：

1. **通过工作区导航栏**：在工作区导航栏底部选择 AIP Assist
2. **键盘快捷键**：
   - macOS：`Cmd+Shift+U`
   - Windows：`Ctrl+Shift+U`

### 界面

AIP Assist 显示在一个面板中，提供：
- 问题输入框
- 对话历史
- 模式选择器

## 使用模式

AIP Assist 提供了几个预配置模式，根据您的工作流程提供更定制的体验：

```
┌─────────────────────────────────────────────────────────────┐
│              AIP Assist 模式                                │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  1. AIP Assist（默认）                                      │
│     ├── 动态选择平台文档                                   │
│     ├── 开发者文档                                         │
│     └── 自定义内容来源                                     │
│                                                             │
│  2. 平台文档辅助（Platform documentation assist）          │
│     └── 根据平台文档回答问题                               │
│                                                             │
│  3. 开发者辅助（Developer assist）                          │
│     ├── 专注于 Foundry APIs                                │
│     └── 常见的开发者示例                                   │
│                                                             │
│  4. AIP Agents                                             │
│     └── 用户开发的、由 LLM 驱动的交互式助手                │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### AIP Assist（默认模式）

**特点**：
- 动态选择多种来源
- 平台文档
- 开发者文档
- 自定义内容来源

**适用场景**：
- 一般问题
- 跨平台查询
- 综合性问题

### 平台文档辅助

**特点**：
- 仅基于平台文档回答
- 权威性高
- 覆盖全面

**适用场景**：
- 平台功能问题
- 操作指导
- 最佳实践

### 开发者辅助

**特点**：
- 专注于 Foundry APIs
- 常见开发者示例
- 代码示例

**适用场景**：
- API 使用问题
- 代码开发
- 集成问题

### AIP Agents

**特点**：
- 用户开发的助手
- 配备企业特定信息
- 仅使用提供的自定义内容来源

**适用场景**：
- 企业特定工作流
- 定制化支持
- 私有知识库

## 自定义内容来源

### 添加自定义来源

您可以添加自定义内容来源来改进 AIP Assist 体验。目前有两种添加自定义内容来源的方法：

#### 1. Notepad 文档（推荐）

- **方式**：使用 Notepad 文档作为内容来源
- **优点**：易于创建和维护
- **适用**：快速添加文档

#### 2. Rosetta 自定义文档

- **方式**：Code Repositories 中 `documentation` 类型的 Markdown 文件
- **优点**：版本控制
- **适用**：正式文档

### 使用自定义来源

一旦创建并注册了自定义来源，有两种方法用其增强 AIP Assist 体验：

```
┌─────────────────────────────────────────────────────────────┐
│        自定义内容来源使用方式                               │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  1. 添加到默认 AIP Assist 知识库                            │
│     ├── 包含在更大的搜索上下文中                           │
│     ├── 与预加载内容混合                                   │
│     └── 适合通用文档                                       │
│                                                             │
│  2. 创建 AIP Agent                                          │
│     ├── 仅使用提供的自定义内容来源                         │
│     ├── 专用的搜索上下文                                   │
│     └── 适合特定工作流                                     │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### AIP Agents vs. 默认知识库

| 特性 | 默认知识库 | AIP Agents |
|------|-----------|------------|
| **搜索上下文** | 预加载 + 自定义 | 仅自定义 |
| **范围** | 广泛 | 专门 |
| **定制化** | 低 | 高 |
| **用例** | 通用文档 | 特定工作流 |
