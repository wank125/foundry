# Vertex 概览

> 来源：https://www.palantir.com/docs/foundry/vertex/overview/

## 概述

**Vertex** 允许您在整个现实世界组织的数字孪生中可视化和量化因果关系。使用 Vertex 工具包，您可以访问和探索组织策划和发布的现有图形，或构建新的系统图形。

Vertex 基于当前、预测或拟议的条件提供数据驱动决策信息，提供了一套工具包来监控、模拟和优化运营决策，以最大化组织成果。

Vertex 可以帮助您：

- **提高透明度**：通过探索跨功能孤岛的整个网络的实时视图
- **关注重要事项**：为顶级风险和机遇及其如何影响目标提供集中警报
- **模拟未来**：结合分析和运营，找到您可以做的最有影响力的改变
- **学习和优化**：通过提高数据清晰度和模型准确性，了解什么有效，并努力实现全局最优
