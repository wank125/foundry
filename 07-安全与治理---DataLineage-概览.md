# Data Lineage 概览

> 更新时间：2025-12-28
> 文档来源：https://palantir.com/docs/foundry/data-lineage/overview/

## 概述

**Data Lineage** 是一个交互式工具，提供数据在 Foundry 平台中如何流动的整体视图。

```
┌─────────────────────────────────────────────────────────────┐
│                     Data Lineage                            │
├─────────────────────────────────────────────────────────────┤
│  数据源 → 数据管道 → 数据集 → Ontology → 工作流 → 应用     │
└─────────────────────────────────────────────────────────────┘
```

## 核心功能

### 1. 查找和发现数据集

- **搜索**：使用 Project、表和列名称搜索数据集
- **浏览**：点击浏览 Foundry Projects 以浏览数据
- **发现**：轻松找到和发现相关数据集

### 2. 探索数据管道

通过强大的界面探索数据管道：

- **展开/隐藏**：展开或隐藏数据集的祖先和后代
- **批量查看**：一次查看一组表的属性
- **可视化**：通过颜色可视化管道（例如，将过期的表着色）
- **深入详情**：深入了解数据详情：
  - 数据架构（Schema）
  - 最后构建时间
  - 生成数据的代码

### 3. 协作

- **管道快照**：创建管道快照与其他用户共享
- **可视化分享**：分享数据流的可视化视图
- **协作分析**：与团队成员协作分析数据流

## 主要用途

### 数据发现

- **搜索数据集**：快速找到所需的数据集
- **理解数据来源**：了解数据从哪里来
- **数据关系**：理解数据集之间的关系

### 管道管理

- **跟踪数据流**：跟踪数据从源头到最终用途的流动
- **影响分析**：了解更改对下游的影响
- **调试管道**：识别和解决管道问题

### 合规和审计

- **数据溯源**：追踪数据的完整历史
- **审计追踪**：满足合规要求的审计追踪
- **透明度**：提供数据处理的透明度

## 图形功能

### 节点着色

- **过期数据集**：将过期的数据集着色
- **权限**：通过着色查看权限
- **健康状态**：可视化数据健康状态
- **自定义着色**：根据特定属性着色节点

### 导航

- **缩放**：放大和缩小图形
- **平移**：在大型图形中导航
- **展开/折叠**：展开或折叠节点的祖先/后代
- **路径高亮**：高亮显示数据流路径

### 节点详情

查看节点的详细信息：
- **架构**：数据集的列和数据类型
- **构建历史**：何时构建数据集
- **代码**：生成数据的转换代码
- **调度**：数据集的调度信息
- **健康检查**：数据健康状态

## 数据集管理

### 管理数据集

- **查看预览**：查看数据集预览和逻辑
- **查看时间线**：查看构建时间线
- **理解过期数据集**：了解哪些数据集过期
- **按列查找**：查找具有给定列的数据集
- **构建数据集**：直接从图形触发构建
- **管理调度**：管理数据集调度
- **回滚**：回滚管道或数据集

### 理解权限

- **检查资源权限**：检查用户查看数据集或工件的权限
- **标记影响**：查看标记更改的影响

## 与其他应用的集成

### Pipeline Builder

- **可视化管道**：可视化在 Pipeline Builder 中构建的管道
- **导航到代码**：导航到生成数据的转换代码

### Ontology

- **探索 Ontology 实体**：探索与数据集相关的 Ontology 实体
- **对象类型链接**：查看数据集如何链接到对象类型

### Workshop

- **应用依赖**：查看 Workshop 应用如何使用数据
- **影响分析**：了解数据更改对应用的影响

## 图形保存和共享

### 保存图形

- **命名图形**：为图形指定有意义的名称
- **组织图形**：将图形组织到文件夹中
- **版本控制**：跟踪图形的更改

### 共享图形

- **URL 共享**：通过 URL 共享图形
- **嵌入**：将图形嵌入到其他应用中
- **导出**：导出图形为图像

## 性能和可扩展性

- **大型管道**：处理包含数千个节点的大型管道
- **快速加载**：快速加载和渲染图形
- **增量更新**：仅更新更改的部分
- **缓存**：缓存频繁访问的数据
