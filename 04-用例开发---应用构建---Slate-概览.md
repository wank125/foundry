# Slate 概览

> 更新时间：2025-12-28
> 文档来源：https://palantir.com/docs/foundry/slate/overview/

## 概述

**Slate** 使应用开发者（构建者）能够使用拖放界面构建具有自定义设计的动态和响应式应用，从而减少开发时间和成本。即使具备最少的编码技能，开发者也能快速创建运营应用、交互式仪表板和自定义登录页面。

## 核心价值

### 主要功能

| 功能 | 说明 |
|------|------|
| **拖放界面** | 直观的拖放界面 |
| **自定义设计** | 完全自定义应用样式 |
| **动态应用** | 构建动态和响应式应用 |
| **快速开发** | 减少开发时间和成本 |
| **CSS 样式** | 使用 CSS 自定义样式 |
| **公共访问** | 构建可公开访问的应用 |

## 样式自定义

### CSS 自定义

使用 CSS 自定义应用样式，您能够：

- **背景颜色**：更改画布背景颜色
- **字体**：自定义字体
- **边框**：调整边框半径
- **布局**：完全控制布局
- **品牌化**：与公司品牌保持一致
- **实验设计**：尝试新设计

**一切皆可改变**：
- 画布背景
- 按钮样式
- 字体和文本
- 边框和间距
- 组件样式

## 公共访问应用

### 功能

Slate 允许构建可被公共互联网访问的应用，无需配置：

- **服务器**：无需服务器配置
- **DNS**：无需 DNS 配置
- **身份验证**：无需身份验证配置

### 数据收集

没有 Foundry 账户的个人可以：

- 上传数据到 Foundry
- 提交表单
- 触发工作流
- 受验证逻辑和安全措施约束

## Ontology 集成

### 无缝利用

Slate 可以无缝利用：

- **对象数据**：使用 Ontology 对象数据
- **Functions**：调用 Foundry Functions
- **Actions**：执行 Ontology Actions
- **外部数据**：集成 Foundry 外部数据源

**外部数据源**：
- 数据库
- API
- Web 服务
- 文件系统
