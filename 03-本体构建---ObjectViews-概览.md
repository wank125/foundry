# Object Views 概览

> 来源：https://www.palantir.com/docs/foundry/object-views/overview/
> 分类：本体构建 / 对象视图

## 概述

**Object View** 是与特定对象相关的所有信息和工作流的中心枢纽。对象视图包括有关对象的关键信息、链接对象、相关指标、分析、仪表板和应用程序。

例如，`Airport` 对象视图（基于美国交通统计局的开源数据）可能提供以下信息：

- 机场名称、城市和位置
- 链接到航班延误行动工作流
- 到达和出发航班的时间线

![机场对象视图信息示例](https://www.palantir.com/docs/resources/foundry/object-views/object-view-overview-example.png)

了解有关对象类型和基于本体的数据建模概念的更多信息。
